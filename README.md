# Onchain Reputation Program (Celo)

## What This Program Does
- Tracks real contributions (building, helping, donating, participating)
- Assigns onchain reputation scores
- Issues soulbound achievement badges
- Displays public user profiles

## Stack
- Solidity + OpenZeppelin
- Celo (Alfajores / Mainnet)
- Next.js + Ethers.js

## Setup
```bash
npm install
npm run compile
npm run dev
